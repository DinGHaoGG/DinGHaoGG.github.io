7.脚注

脚注的格式为: [^要注明的文本]

8.列表

1.无序列表,在前面加*星号、+加号、-减号都可以

上面的语法的显示如下:

  • 第一项

  • 第三项

2.有序列表，使用数字并加上.点号

上面的语法的显示如下:

上面的语法的显示如下:

  1. 第一项
    • 第一项的第一个元素
    • 第二个元素
  2. 第二项
    • 第二项的第一个元素
    • 第二个元素

}

11.链接

[链接名称] (链接地址) 或者 <链接地址>

上面的语法的显示如下:
这是百度的链接
https://www.baidu.com

高级链接的使用

这个链接用1作为网址变量[百度][1]
[1]:https://www.baidu.com

12.图片

使用 ![alt 属性文本] (图片地址)

其中alt是图片的替代文字

上面的语法的显示如下:
其中alt是图片的替代文字

如果需要设置图片的宽高可以使用HTML中的<img>标签

12.表格

使用|来分隔不同的单元格,使用-来分隔表头和其他行

上面的语法的显示如下:

表头 表头
